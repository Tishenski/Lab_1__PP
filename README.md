# Інструкція до запуску програми:

## Програма "Генератор паролів"

Проста програма на Python, яка генерує випадковий пароль заданої довжини.

## Як запустити

1. Переконайтеся, що у вас встановлено Python (версія 3 або новіша).
   Перевірити можна командою: python --version

2. Завантажте або клонуйте репозиторій:
git clone https://github.com/AlexBerlimenko/Lab_1__PP.git

3. Перейдіть до папки проекти: cd Lab_1__PP

4. Запустіть програму: python main.py

5. Введіть бажану довжину пароля (не менше 4 символів).
